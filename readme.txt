Use your email templates for your Give donations

== Description ==
A simple add-on to use your [WP HTML Mail](https://wordpress.org/plugins/wp-html-mail/) email template with [Give](https://wordpress.org/plugins/give/) donations. 

I also works with [per form emails add-on](https://givewp.com/addons/per-form-emails/) for give.

Simply design your email template without writing a single line of code.
One template for all your WordPress emails. Works with WordPress core emails, several contact form plugins, WooCommerce, Easy Digital Downloads ...

... and it's free

https://youtu.be/RV8vUNCWOZw

[read more](https://wordpress.org/plugins/wp-html-mail/)


== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
